= /dev/payments ruby bindings

== Installation

You don't need this source code unless you want to modify the gem. If you just want to use the /dev/payments ruby bindings, you should run:

  curl -o devpayments.gem http://devpayments.com/bindings/ruby/devpayments-latest.gem
  sudo gem install rest-client json
  sudo gem install devpayments.gem

If you want to build the gem, first install hoe. Then do:

  sudo rake install_gem

== REQUIREMENTS:

* rest-client, json
