= /dev/payments ruby bindings

== Installation

You don't need this source code unless you want to modify the gem. If you just want to use the /dev/payments ruby bindings, you should run:

  sudo gem install --source http://gems.devpayments.com devpayments

If you want to build the gem from source, first install hoe. Then do:

  sudo rake install_gem

== REQUIREMENTS:

* rest-client, json
